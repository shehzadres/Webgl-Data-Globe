# [1.0.0] - Initial Production Release

## Added

### Rendering

- Realistic physically-inspired Earth rendering
- Day/night texture compositing
- Normal-mapped terrain
- Specular ocean highlights
- Atmospheric glow shader

### Camera

- Custom spherical-coordinate camera engine
- Smooth interpolation
- Cinematic camera presets
- Camera focus system

### Visualization

- Interactive geographic markers
- Data-driven spike visualization
- Billboard labels
- Great-circle flight routes

### Particles

- Star field
- Orbital particle system
- Ambient space particles
- Centralized particle manager

### Shaders

- Atmosphere shader
- Glow shader
- Procedural noise shader
- GPU star shader
- Shared shader utilities

### User Interface

- Scene Director
- Scroll-driven storytelling
- Search
- HUD
- Tooltips
- Control panel
- Scene navigation

### Performance

- BufferGeometry particle rendering
- Shared shader uniforms
- Cached texture loading
- Adaptive DPR
- Zero per-frame heap allocations
- Optimized render loop

### Tooling

- TypeScript
- ESLint
- Prettier
- GitHub Actions CI
- Vite
- Embeddable widget
